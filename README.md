# Minimalist

Tiny boilerplate for building fast and modern SPA's! ‚ù§

### Installing

```
git clone https://github.com/rabyyuson/minimalist.git
cd minimalist
yarn install
```

### Running

Development mode:

```
yarn start
```

Production build:

```
yarn build
```

GraphQL endpoint:

```
http://localhost:8080/graphql
```
