# Minimalist

A tiny boilerplate for start your awesome modern Single Page Application (SPA)!

### Installing

```
git clone https://github.com/rabyyuson/minimalist.git
cd minimalist
yarn install
```

### Running

To run it on dev with HMR:

```
yarn start-dev
```

To run it on normally (build for production):

```
yarn start
```
