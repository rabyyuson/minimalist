# Minimalist

Starting web application made easy with this tiny starter framework.

### Installing

```
git clone https://github.com/rabyyuson/minimalist.git
cd minimalist
yarn install
```

### Running

To run it on dev with HMR:

```
yarn start-dev
```

To run it on normally (build for production):

```
yarn start
```
