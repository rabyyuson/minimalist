# Minimalist

A tiny boilerplate for starting modern SPAs.

### Installing

```
git clone https://github.com/rabyyuson/minimalist.git
cd minimalist
yarn install
```

### Running

To run it on development mode with HMR:

```
yarn start-dev
```

To run it on normally:

```
yarn start
```
